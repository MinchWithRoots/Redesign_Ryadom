# 🛡️ Админ Панель - Руководство

## Что это такое?

Админ панель - это инструмент управления платформой, доступный только администраторам. Позволяет:
- 📊 Просматривать статистику (пользователей, консультантов, чатов, отзывов)
- 👥 Управлять пользователями (удалять, менять роли)
- 🎯 Управлять консультантами (включать/отключать)
- ⭐ Модерировать отзывы (удалять)
- 💬 Просматривать историю чатов

## 🔐 Как сделать пользователя администратором?

### Способ 1: Через Supabase Console (Рекомендуется)

1. Перейди на https://app.supabase.com
2. Выбери свой проект
3. Перейди на **SQL Editor** (левое меню)
4. Выполни этот SQL запрос, заменив `user@example.com` на нужный email:

```sql
UPDATE users 
SET role = 'admin' 
WHERE email = 'user@example.com';
```

5. Нажми **Run** (Ctrl+Enter)
6. Пользователь теперь администратор! ✅

### Способ 2: Через Table Editor

1. В Supabase перейди на **Table Editor**
2. Открой таблицу **users**
3. Найди нужного пользователя
4. Измени поле `role` на `admin`
5. Сохрани изменения

### Способ 3: Через API (для разработчиков)

```javascript
import { supabase } from '@/utils/supabase'

// Сделать пользователя админом
const { data, error } = await supabase
  .from('users')
  .update({ role: 'admin' })
  .eq('email', 'user@example.com')
  .select()

if (error) {
  console.error('Ошибка:', error)
} else {
  console.log('Пользователь назначен админом:', data)
}
```

## 📍 Где находится админка?

1. **Для администраторов:** В профиле есть кнопка "🛡️ Админ панель"
2. **Прямая ссылка:** `/admin`
3. **Доступ:** Только авторизованным администраторам

## 💻 Функционал админки

### 1. Вкладка "Обзор"
- Информация о возможностях панели
- Статистика по основным показателям

### 2. Вкладка "Пользователи"
- Список всех пользователей
- Возможность сделать пользователя админом или убрать права
- Удаление пользователя (недоступно для админов)

### 3. Вкладка "Консультанты"
- Список всех консультантов
- Включение/отключение доступности
- Просмотр рейтинга и отзывов

### 4. Вкладка "Отзывы"
- Все опубликованные отзывы
- Удаление отзывов при необходимости

### 5. Вкладка "Чаты"
- История всех чатов на платформе
- Статус (активный/завершенный)
- Количество сообщений

## ⚙️ Технические детали

### Структура данных

**Таблица users:**
```
id          | name     | email           | role
------------|----------|-----------------|--------
1           | Иван И.  | ivan@example.com| admin
2           | Мария К. | maria@ex.com    | user
```

**Важно:** Роль `admin` дает полный доступ к админ панели!

### Защита

- Маршрут `/admin` защищен от неавторизованных пользователей
- Только пользователи с `role = 'admin'` могут войти
- Все действия логируются в Supabase

## 🚀 Примеры использования

### Пример 1: Создать первого админа

```sql
-- Если у тебя уже есть пользователь в БД
UPDATE users 
SET role = 'admin' 
WHERE id = 1;

-- Или просмотри всех пользователей
SELECT id, name, email, role FROM users;
```

### Пример 2: Получить всех админов

```sql
SELECT * FROM users WHERE role = 'admin';
```

### Пример 3: Убрать админ права

```sql
UPDATE users 
SET role = 'user' 
WHERE email = 'admin@example.com';
```

⚠️ **Важно!**
- Роль админа дает полный доступ к панели управления
- Выдавай админ права только надежным людям
- Регулярно проверяй список админов
- Если админ больше не нужен - убери его права

## 🆘 Решение проблем

### "Кнопка админки не видна"
- ✅ Проверь, что пользователю установлена роль `admin` в Supabase
- ✅ Перезагрузи страницу (F5)
- ✅ Очисти кэш браузера (Ctrl+Shift+Delete)

### "Админка не загружается"
- ✅ Проверь подключение к Supabase (переменные окружения)
- ✅ Убедись, что таблицы созданы в БД
- ✅ Проверь консоль браузера (F12) на ошибки

### "Не могу удалить админа"
- ✅ Админы защищены от удаления (разумная безопасность)
- ✅ Убери админ права сначала, потом удали
