## 🔐 Безопасность

### Проверки доступа

1. **При загрузке страницы /admin:**
   ```typescript
   if (!isAdmin()) {
     router.push('/') // Редирект обычного пользователя
   }
   ```

2. **При каждом действии:**
   ```typescript
   try {
     const { error } = await supabase.from('users')...
     if (error) throw error
     // Success message
   } catch (err) {
     // Error message
   }
   ```

3. **Защита на уровне маршрутизации:**
   ```typescript
   {
     path: '/admin',
     meta: { requiresAuth: true, requiresAdmin: true }
   }
   ```
