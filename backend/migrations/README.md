# Database Setup - Companions

Инструкция по добавлению тестовых данных спутников.

**ВАЖНО:** Это не профессиональный сервис психологов. Это платформа для людей, которые помогают друг другу оставаться в терапии.

## Структура таблицы companions

```sql
- id: BIGINT (primary key)
- name: TEXT
- age: INT
- gender: TEXT (female/male)
- experience: TEXT (beginner/experienced/expert)
- bio: TEXT (краткое описание человека)
- image: TEXT (ссылка на фото)
- specialization: TEXT (темы разговоров, через запятую)
- reviews_count: INT
- created_at: TIMESTAMP
- updated_at: TIMESTAMP
- is_available: BOOLEAN
```

## Миграции

1. `001-add-specialization.sql` - добавляет поле `specialization` если его нет
2. `002-seed-companions-with-specializations.sql` - добавляет тестовых спутников

## Как применить

1. Откройте [Supabase Dashboard](https://app.supabase.com)
2. Перейдите в **SQL Editor**
3. Скопируйте и выполните скрипты по порядку

## Frontend

Фильтр по темам разговоров добавлен в `src/views/SearchView.vue` как `specialization`.

Динамически загружает темы из поля `specialization` каждого спутника.

## Проверка

1. Откройте `/search`
2. В фильтрах должны появиться темы (Депрессия, Тревога и т.д.)
3. Выберите тему - список отфильтруется
4. "Сбросить фильтры" вернёт всех
