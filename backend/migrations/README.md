# Database Migrations

Инструкция по применению миграций базы данных для добавления специализаций спутников.

## Структура

- `001-create-specializations-table.sql` - Создает таблицу `companion_specializations`
- `002-seed-companions-with-specializations.sql` - Добавляет тестовые данные

## Как применить

### Шаг 1: Создание таблицы специализаций

1. Перейдите в [Supabase Dashboard](https://app.supabase.com)
2. Выберите ваш проект
3. Откройте меню **SQL Editor**
4. Нажмите **New Query**
5. Скопируйте содержимое файла `001-create-specializations-table.sql`
6. Нажмите **Run**

Ожидаемый результат:
```
Table 'companion_specializations' created successfully
```

### Шаг 2: Добавление тестовых данных

1. В SQL Editor создайте новый запрос
2. Скопируйте содержимое файла `002-seed-companions-with-specializations.sql`
3. Нажмите **Run**

Ожидаемый результат:
```
Migration completed!
total_companions: 8
total_specializations: 24
total_topics: 24
```

## Структура таблицы companion_specializations

```sql
CREATE TABLE companion_specializations (
  id BIGSERIAL PRIMARY KEY,
  companion_id BIGINT NOT NULL REFERENCES companions(id) ON DELETE CASCADE,
  specialization TEXT NOT NULL,
  created_at TIMESTAMP DEFAULT NOW(),
  UNIQUE(companion_id, specialization)
);
```

## Код для фронтенда

Виджет фильтрации по специализации уже добавлен в `src/views/SearchView.vue`.

Фильтр динамически загружает все уникальные специализации из базы данных и отображает их как радио-кнопки в боковой панели фильтров.

### Как работает фильтрация

1. Функция `loadCompanions()` в `src/composables/useAppState.ts` загружает спутников вместе со специализациями
2. Компонент `SearchView.vue` вычисляет уникальные специализации через `computed` свойство
3. UI отображает специализации как радио-кнопки
4. При выборе специализации применяется фильтр к списку спутников

## Проверка

Для проверки того, что всё работает:

1. Откройте страницу поиска `/search`
2. В боковой панели фильтров должны появиться специализации
3. Попробуйте выбрать специализацию - список спутников должен отфильтроваться
4. Нажмите "Сбросить фильтры" - должны появиться все спутники

## Если что-то пошло не так

Если вы хотите очистить и начать заново:

```sql
-- Удалить все записи о специализациях
DELETE FROM companion_specializations;

-- Удалить все спутников
DELETE FROM companions;

-- Затем повторить Шаг 2
```
