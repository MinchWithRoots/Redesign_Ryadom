# Инструкция: Исправление отображения аватарок спутников

## Проблема
Аватарки спутников не отображались правильно в поиске спутников.

## Исправления, которые были сделаны

### 1. **Расширены лимиты для URL изображений**
   - Изменены поля `image` в таблицах `companions`, `companion_applications` и `users` с `VARCHAR(500)` на `TEXT` (без лимита)
   - Это позволит сохранять длинные URL изображений из Supabase Storage

**Выполните в Supabase SQL Editor:**
```sql
-- Increase the image field limit from VARCHAR to TEXT (no limit)
ALTER TABLE public.companions ALTER COLUMN image TYPE TEXT;
ALTER TABLE public.companion_applications ALTER COLUMN image TYPE TEXT;
ALTER TABLE public.users ALTER COLUMN image TYPE TEXT;
```

### 2. **Добавлена фильтрация в поиск**
   - Функция `loadCompanions()` теперь загружает только доступных спутников (is_available = true)
   - Файл: `src/composables/useAppState.ts`

### 3. **Добавлена загрузка изображения в форму заявки**
   - Пользователи теперь могут загрузить свою фотографию при подаче заявки на становление спутником
   - Поддерживаются файлы до 5MB
   - Файл: `src/views/CompanionApplicationView.vue`

### 4. **Улучшена обработка изображений при одобрении**
   - При одобрении заявки администратор, изображение корректно сохраняется в таблицу `companions`
   - Файл: `src/views/AdminDashboardView.vue`

## Последовательность действий

1. **Сначала выполните SQL-миграцию в Supabase** (см. выше)
2. **Протестируйте форму заявки:**
   - Перейдите на страницу "Стань спутником"
   - Загрузите фотографию через новое поле
   - Заполните остальную информацию и отправьте заявку
3. **Протестируйте админ-панель:**
   - Откройте админ-панель → вкладка "Заявки"
   - Одобрите заявку
   - Проверьте что спутник появился в списке доступных
4. **Проверьте поиск:**
   - Откройте страницу поиска спутников
   - Проверьте что аватарки отображаются корректно

## Что изменилось для пользователей

- ✅ Возможность загрузить фотографию при подаче заявки на становление спутником
- ✅ Аватарки теперь корректно отображаются в поиске спутников
- ✅ Поддерживаются длинные URL изображений из облачного хранилища
